## ✨ 功能

### 🍳 核心烹饪功能
- **食谱数据库**：
  - 🌟 无限量食谱
  - 详细视图：
    - 🛒 食材清单
    - 📝 个人笔记
    - 📹 嵌入式教程
    - ⏲️ 烹饪指导
    - 📊 营养信息（卡路里、蛋白质等）
    - ⚠️ 过敏原警告（坚果、麸质等）
    - 📸 成品照片
    - 🗂️ 分类标签（素食、快手菜等）
    - 📅 准备时间
    - 🔄 分量调整

### 🛒 智能购物清单
- 🛒自动从食谱生成
- 📶离线访问
- 🏷️智能分类：
  - 🥦 新鲜蔬果
  - 🍞 烘焙食品
  - 🍗 肉禽海鲜
  - 🥫 包装食品
  - 🧂 调味品
- 💰 价格追踪
- 🗒️ 手动添加物品
- 🗑️ 标记已购买物品
- 🔄 与家人共享清单

### 🧠 高级功能
  - 🗓️ 每周拖拽式排餐计划
  - 🍽️ 智能餐食推荐：
	- 🥗 饮食偏好
	- 🥦 现有食材
	- 🍽️ 烹饪时长
  - 📊 每周计划营养分析
  - 🧹 "清空冰箱"模式（根据临期食材推荐食谱）
  - 🌱 饮食限制（素食、生酮等）
  - 💰 食材采购预算追踪
  - 👩‍🍳 烹饪难度等级（新手/进阶）
