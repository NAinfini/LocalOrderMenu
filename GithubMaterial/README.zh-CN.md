## ✨ 功能

### 🍳 核心烹饪功能
- **食谱数据库**：
  - 🌟 无限量食谱
  - 详细视图包含：
    - 🛒 带份量的食材清单
    - 📝 个人笔记
    - 📹 嵌入式YouTube教程
    - ⏲️ 分步烹饪指导
    - 📊 营养信息（卡路里、蛋白质等）
    - ⚠️ 过敏原警告（坚果、麸质等）
    - 📸 成品照片
    - 🗂️ 分类标签（素食、快手菜等）

### 🛒 智能购物清单
- 自动从食谱生成
- 智能分类：
  - 🥦 新鲜蔬果
  - 🍞 烘焙食品
  - 🍗 肉禽海鲜
  - 🥫 包装食品
  - 🧂 调味品
- 支持多人同步协作

### 🧠 高级功能
- AI智能推荐食谱
- 冰箱清空模式
- 饮食偏好设置
- 预算追踪功能

- MVVM架构

- Firebase Firestore
- Room数据库
